There should be a config.ini in this directory when you run it.
If it does not exist, it will automatically generate this file and then exit, Run it again to then launch the server
That file contains most of the configuration options for the server

If excess clients try to connect, server will not respond

For windows (Installer based installation):

If the registry keys for this program are ever broken setup.exe should fix it. And uninstall.exe will remove them.
The only place where files are saved is the program files directory
